Пособие для использования приложения

Для работы с файлом XML необходимо его загрузить нажав на кнопку "Загрузить файл XML". После выбора файла загрузятся dom ветки файла, которые были выбраны для вывода в элемент древовидного представления данных. Для сохранения данных выберите узлы в элементе древовидного представления данных и нажмите на кнопку "Сохранить выбранное", напишите имя файла и нажмите сохранить
